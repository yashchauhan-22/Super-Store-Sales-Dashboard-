# 📊 SuperStore Sales Dashboard | Power BI

## 📌 Project Overview
This project presents an interactive **SuperStore Sales Dashboard** built using **Power BI**. The dashboard provides insights into sales performance, profit trends, customer segments, shipping modes, and future sales forecasting to support data-driven business decisions.

## 🎯 Objectives
- Analyze overall sales, profit, and order performance.
- Identify top-performing product categories and sub-categories.
- Compare monthly sales and profit trends year-over-year.
- Understand customer purchasing behavior by segment and payment mode.
- Evaluate regional performance and shipping preferences.
- Forecast future sales for better business planning.

## 🛠️ Tools & Technologies
- **Power BI Desktop**
- Data Visualization
- Data Modeling
- DAX (Data Analysis Expressions)
- Forecasting Analytics

## 📈 Dashboard KPIs
- **Total Sales:** 2M+
- **Total Orders:** 22K+
- **Total Profit:** 175K+
- **Average Ship Days:** 4 Days

## 📊 Key Insights

### Sales by Category
- Office Supplies: 0.64M
- Technology: 0.47M
- Furniture: 0.45M

### Top Sub-Categories
- Phones: 0.20M
- Chairs: 0.18M
- Binders: 0.17M

### Customer Segments
- Consumer: 48%
- Corporate: 33%
- Home Office: 19%

### Payment Modes
- Cash on Delivery (COD): 43%
- Online Payments: 35%
- Cards: 22%

### Shipping Modes
- Standard Class: Highest usage
- Second Class
- First Class
- Same Day

## 🔮 Sales Forecasting
The dashboard includes a **15-Day Sales Forecast** feature that helps predict future sales trends based on historical data, enabling better inventory and business planning.

## 🌍 Regional Analysis
The dashboard provides sales distribution across:
- Central
- East
- South
- West

along with shipping mode performance by region.
